# AGENTS.md

このリポジトリは、利用者が日常作業に使う CLI 作業用コンテナのコードです。

## 作業環境の前提

- Codex は devcontainer 内で起動されることがある。
  - Codex 側から `sudo` の利用は避ける。sudo パスワード入力を前提にした調査や検証は行わない。
- Docker daemon / containerd socket へのアクセスが権限不足になる場合がある。その場合は `sudo docker run` などに進まず、権限不足として扱い、必要ならホスト側で実行するコマンドを提示する。
- `docker manifest inspect` のように Docker daemon へ接続せずに済む確認は利用してよい。
- 破壊的操作、ホストランタイムへ影響する操作、root 権限が必要な操作は、read only な代替調査を先に行う。

## 調査方針

- ドキュメントの更新が必要な箇所を見つけた場合は、修正するか、カレントリポジトリ外であれば提案としてまとめる。
- 不明点は read only な調査を先に尽くし、実行しないと分からない事項は不確定として明示する。

## 説明方針

- このシステムは自宅の検証環境として扱い、リスク評価は家庭向け実験用の前提で行う。
- システム構成や Ansible がシンプルになる余地があれば、全体構成への影響も含めて提案する。
